About Glen

This is my about page. When you log into DVO, a file called about.html will be created in a folder called DVO, along with two other folders.  One is for posts, like the ones below, and the other is for comments. The comments folder is for the DVO chrome extension. The DVO extension allows you to like, dislike and comment on any URL. It uses GunDB to aggregate the comments from other users. GunDB is a distributed database, written entirely in JavaScript. By default, data is stored in localStorage, and is replicated to other peers who are using your application. It uses a sophisticated conflict resolution algorithm to achieve eventual consistency, and the syntax is similar to Firebase.
